# Postgresql (Recommended database)

## Install

See the [official documentation](https://www.postgresql.org/docs/9.3/tutorial-install.html) on how to get started to install Postgresql.
You can either install on your system directly, with docker, or using a cloud provider.

## Databases

We recommend creating and using the following database names:

- **django**: Used by **celery-admin**.
- **grafana**: Used by **grafana** service.
- **www_api**: Use by **www-api**.

# loki

Loki is responsible for storing logs and processing queries from Grafana.

## Booting it up

```sh
sh init_env.sh
docker-compose up -d loki
```

## Shutdown

```sh
sh init_env.sh
docker-compose stop loki
```

## Configurations

The configurations should be located in: grafana/loki/etc/local-config.yaml

You can copy some sample configuration with:

```sh
cp grafana/loki/etc/local-config.yaml.sample grafana/loki/etc/local-config.yaml
```
